This is a very minimal library - 29 lines to easily add 9 colors to your terminal output.

Here's how to use it:

```ts
import { green } from 'discolor';

console.log(green('I will be green'));
```

Possible colors you can use:

1. black
2. red
3. green
4. yellow
5. blue
6. magenta
7. cyan
8. white
9. gray
